# Calculator Project

## Description

This project is a simple yet functional web-based calculator. It demonstrates the use of HTML for structure, CSS for styling, and JavaScript for interactive functionality. The calculator can perform basic arithmetic operations like addition, subtraction, multiplication, and division.

## Features

- Basic arithmetic operations: addition, subtraction, multiplication, division
- Clean and intuitive user interface
- Responsive design for various screen sizes
- Keyboard support for desktop users

## Technologies Used

- HTML
- CSS
- JavaScript
